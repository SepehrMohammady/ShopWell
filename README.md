## 🛒 What is ShopWell?

ShopWell is a free, offline-first shopping app that helps you **organize your shopping life in one place**. Whether you're managing weekly groceries, comparing prices across stores, or planning your next shopping trip, ShopWell keeps everything tidy and at your fingertips — no account or internet connection required.

---

## ✨ Features at a Glance

| Feature | Description |
|---|---|
| 📝 **Shopping Lists** | Create and manage multiple shopping lists. Track what you need, what you have, and never forget an item again. |
| 🏪 **Shop Directory** | Save your favorite shops with their categories, addresses, and notes. Quickly find where to go. |
| 💰 **Price Comparison** | Add product prices at different shops and brands. ShopWell tells you where to get the best deal. |
| 🛍️ **Shop Mode** | Open a shop and see all your products available there, complete with price warnings when something is cheaper elsewhere. |
| 📅 **Schedule Trips** | Plan your shopping trips in advance with reminders. Link a schedule to a specific list and shop. |
| 📦 **Product Availability** | Mark items as *Need to Buy* or *Already Available* so you always know what's in your pantry. |
| 🏆 **Best Shop Picks** | ShopWell recommends the best shop for your current list — showing how many items are available and the estimated total cost. |
| 🌙 **Dark Mode** | Comfortable on the eyes day or night. Choose between Light, Dark, or System theme. |
| 💾 **Fully Offline** | All your data is stored privately on your device. No account. No cloud. No tracking. |

## 🚀 Getting Started

> **Just want to use the app?** Download the latest APK from the [Releases](https://github.com/SepehrMohammady/ShopWell/releases) page and install it on your Android device.

<details>
<summary>🔧 Build from source (developers)</summary>

### Prerequisites
- Node.js >= 18
- Android Studio (for Android), Xcode on macOS (for iOS), or Visual Studio (for Windows)

### Setup

```bash
# 1. Clone the repo
git clone https://github.com/SepehrMohammady/ShopWell.git
cd ShopWell

# 2. Install dependencies
npm install

# 3. (iOS only) Install CocoaPods
cd ios && pod install && cd ..
```

### Run

```bash
npm run android   # Android
npm run ios       # iOS
npm run windows   # Windows
```

</details>

## 🗺️ How It Works

```
1. Add your Shops          →  Save the stores you visit regularly
2. Build your Products     →  Add items with prices at each shop
3. Create a Shopping List  →  Pick what you need to buy
4. Check Best Shop Picks   →  ShopWell recommends where to shop & total cost
5. Start Shop Mode         →  Walk through the store and check items off
6. Schedule Future Trips   →  Set a reminder for your next run
```
